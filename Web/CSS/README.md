# CSS Quick Guide

## ❓ What is CSS?
- Cascading Style Sheets
- Styles HTML elements
- Controls layout and presentation

## 🔑 Key Features
- Selectors (class, ID, element)
- Box model
- Flexbox/Grid layouts
- Responsive design
- Animations

## 💻 Run Examples
### 1. Link to HTML file
```html
<link rel="stylesheet" href="explanation.css">
```
### 2. View in browser

## 🌐 Common Uses
- Website styling
- Responsive designs
- Animations
- Print styling

1. Basic selectors
2. Box model
